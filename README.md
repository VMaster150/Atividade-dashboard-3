# Atividade-dashboard

1 - Na aba de seleção coloque o que é desejado, (Primeiro: Data, Segundo: Tipo, Terceiro: Valor, Quarto: Categoria, Quinto: Descrição);

2 - Ao gerar o seu depósito ou despesa é criado um gráfico com esses dados coletados;

3 - É gerado também um relatório geral e em ordem de implementação dos mais recentes até o mais antigo;

4 - Filtragem de de dados por depósito ou despesas;
